## コンパイル

コンパイルにはLuaLaTeXが必要です。

- [TeXLive](https://www.tug.org/texlive/)（または[MacTeX](https://tug.org/mactex/)）をインストール
- `make`を実行

## アニメーションの削除

`documentclass`に`handout`を足すと、アニメーションを削除できます。
